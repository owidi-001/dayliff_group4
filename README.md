## Set up
- Pull from the main repo:
`git clone git@github.com:owidi-001/dayliff_group4.git`

- Set env variables
  `cd dayliff_group4`
  `cat >> .env`
  `MAPS_KEY= <MAPS KEY VALUE>`
  
- Run the app
  `flutter run -v`
